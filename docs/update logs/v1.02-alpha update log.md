# v1.02-alpha Update Log

Baseline used for this summary: `v1.01`, the existing git tag for the previous release.

Endpoint used: `ab51d3b` from 2026-05-21, the commit labeled `v1.02-alpha release`.

## Highlights

- Added transient-solver support and an animation display path for time-varying results.
- Added optional convection-term solving and more convection discretization choices.
- Added Red-Black Gauss-Seidel as a linear solver option.
- Improved SIMPLE solver stability with bug fixes around divergence and transient behavior.
- Reworked the results inspector so field coloring is shader-based instead of CPU-generated.
- Added zooming, panning, value probing, and filtering to the 2D inspector and colorbar workflow.
- Overhauled result visualization with flat/interpolated shading options and timestep-aware 3D model updates.
- Expanded solver and residual UI support, including multiple dockable residual plots and residual plot toolbars.
- Added fluid-property editing with unit support and improved mesh-variable unit handling.
- Added PNG-backed icon/texture loading and new result/inspector preview screenshots.

## Solver And Numerics

- Added an option to include the convection term in the solver.
- Added transient solver handling and time-dependent result support.
- Added fully developed flow as an inlet axial-velocity option.
- Added Red-Black Gauss-Seidel solver support.
- Fixed a SIMPLE algorithm bug and later fixed a divergence issue.
- Added convection discretization options for central difference and second-order upwind.
- Fixed bugs affecting transient solver behavior.
- Added editable fluid properties with user-selectable units.
- Added unit support for mesh variables and save/load support for units.

## Results And Visualization

- Added an animation GUI/class for displaying transient results.
- Updated the 3D model based on timestep and selected result variable.
- Optimized GPU usage by reducing the number of instances drawn to the screen.
- Added uniform caching.
- Added flat and interpolated shading options for results.
- Added PNG texture loading through `TextureBuffer`, mainly for button icons.
- Added new shader files for the inspector.
- Updated result shaders and result-view rendering paths.

## Inspector And Colorbar

- Reworked the inspector to color fields through shaders instead of generating textures on the CPU.
- Added zooming and panning to the inspector.
- Added hover dots at the mouse position.
- Added point/value readouts that adapt to zoom and pan.
- Added colorbar-click filtering.
- Added between/exclude filter modes.
- Merged colorbar and inspector behavior more tightly.
- Added copy-to-clipboard support for inspector output.
- Added a select-region inspector icon.

## Solver UI And Residuals

- Made major solver UI changes around transient and solver-option workflows.
- Added more residual plot functionality.
- Added support for multiple residual plots in the solver UI.
- Added docking support for residual plot panels.
- Added residual plot toolbars.
- Added clipboard support for residual plots through the `clip.h` library.

## Mesh, Units, And UI

- Made a major mesh-generation update.
- Overhauled the mesh GUI.
- Added mesh-variable units and unit persistence.
- Added a `Unit` namespace to avoid naming conflicts.
- Added `initContext` initialization in GUI setup.
- Reorganized mouse handling by moving away from GLFW callbacks and into ImGui-based callbacks.
- Added `gui_manager.h` and related GUI resource management.

## Documentation And Release Materials

- Updated the README several times.
- Added v1.02 preview screenshots.
- Renamed the software manual PDF to `docs/AxiSim_Software_Manual v1.01-alpha.pdf`.
- Updated documentation/image folders for v1.0, v1.01, and v1.02 screenshots.

## Known Work-In-Progress Notes From This Range

- One commit notes that SIMPLE may still need bug fixing after the Red-Black Gauss-Seidel work.
- One commit notes an incorrect color-assignment issue in the timestep-based 3D model path.
- The release commit is labeled `v1.02-alpha`, but the README at that exact commit still reports `Current Version: v1.01`.
