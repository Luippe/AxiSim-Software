# Main Changes Since v1.03-alpha

Baseline used for this summary: `fc18e8a` from 2026-06-14, the commit where the README identifies the project as `v1.03-alpha`.

Current endpoint used: `96e198b` from 2026-07-05. Also included below are working-tree changes from 2026-07-06 that are not yet committed (console autocomplete and colorbar auto-sizing); these are marked as uncommitted in the Commit Reference.

Note: there is no `v1.03-alpha` git tag in this repository, so this file treats the README version bump commit as the v1.03-alpha baseline.

## Highlights

- Added the new geometry/sketch workflow for building axisymmetric cross-sections directly in the app.
- Added Gmsh-backed unstructured triangular meshing, including boundary discretization, boundary grouping, and region-of-influence refinement.
- Added support for using structured and unstructured meshes side by side, with sketch-to-structured and sketch-to-unstructured conversion paths.
- Added concentration transport, new concentration boundary conditions, Michaelis-Menten and Hill kinetics, and wall-consumption support.
- Reworked save/load behavior so project state is persisted together across geometry, mesh, solver, and results-related data.
- Added updated v1.04-alpha documentation, screenshots, packaging support, and licensing notes for bundled dependencies.
- Added a command-console autocomplete dropdown and made the results colorbar auto-size so its labels and values are never clipped (uncommitted, 2026-07-06).

## Geometry And Sketching

- Introduced a dedicated geometry/sketch view.
- Added sketch entities for lines, rectangles, circles, arcs.
- Added selection and move support for sketch geometry.
- Added snapping and trim-related sketch tooling.
- Added sketch toolbar icons and sketch shaders.
- Added sketch-to-mesh conversion so drawn geometry can drive mesh generation.
- Added quality-of-life fixes and interaction improvements in the geometry tab.

## Meshing

- Added unstructured mesh support using bundled Gmsh.
- Added Gmsh constrained triangulation from sketch boundaries.
- Added boundary vertices, boundary edges, boundary segments, and boundary segment groups for unstructured meshes.
- Added per-boundary sizing controls, including edge count, target spacing, and bias.
- Added region-of-influence refinement controls with circular and rectangular regions.
- Added finite-volume mesh creation for unstructured triangles.
- Added unstructured mesh rendering and inspection paths.
- Added support for structured mesh generation from sketches by rasterizing the sketch into a Cartesian grid.
- Added support for choosing between structured and unstructured mesh types.
- Added logic to keep boundary groups usable across mesh regeneration where possible.
- Added obstacle/domain boundary handling for mesh inspection and boundary assignment.

## Solver And Physics

- Added optional concentration solving through `solveConcentration`.
- Added concentration equation assembly and solution code.
- Added concentration residual plotting and solution-field storage.
- Added concentration boundary data upload to the GPU solver path.
- Added Michaelis-Menten and Hill boundary condition types.
- Added optional substrate-inhibition parameters for kinetic boundary conditions.
- Added wall-consumption behavior for concentration.
- Added diffusion-layer data structures for concentration and temperature wall layers.
- Added fixes to the concentration solver after initial implementation.
- Fixed inhibition so it does not affect the solution field when inhibition is disabled.
- Extended boundary-condition data structures to cover velocity, pressure, static temperature, and concentration more consistently.

## Results And Inspection

- Reworked the shared surface viewer infrastructure used by inspectors.
- Updated inspectors and mesh views to support the new mesh and geometry data paths.
- Added mesh cell inspection helpers, including non-orthogonality reporting for inspected cells.
- Added support for inspecting unstructured finite-volume data.
- Added result shaders for unstructured-result rendering paths.
- Kept residual plotting aligned with the added concentration field.
- Replaced the external `clip.h` dependency with local clipboard handling.
- Added automatic sizing of the results colorbar so its tick values and field label always fit and are never clipped by the panel edge.

## Console And Command Input

- Added a live autocomplete dropdown to the command console that appears above the input line as you type, instead of printing candidates into the log.
- Added action, object, and value completion (for example `set colormap <name>`), driven by each command's registered objects and per-object value lists.
- Added keyboard control for the dropdown: Up/Down to move the highlight and Tab, Enter, or Right Arrow to accept; command history still works when the dropdown is closed.

## Save, Load, And Project State

- Reworked project save/load so projects are saved as a whole instead of separate mesh, solver, and results files.
- Added persistence for geometry/sketch state.
- Added persistence for mesh type, mesh data, boundary groups, boundary sizing, and regions of influence.
- Added persistence for solver options and expanded boundary-condition parameters.
- Added file-versioning support in the binary project format.
- Added launch-time project/settings loading paths used by the current workflow.

## Build, Packaging, And Dependencies

- Added bundled Gmsh files and CMake integration for unstructured mesh generation.
- Removed the `tinyfiledialogs` dependency.
- Removed the `clip-main` dependency.
- Added a local Windows clipboard helper for text and image copying.
- Added `package.ps1` for building and assembling redistributable AxiSim ZIP packages.
- Updated CMake install/package rules for runtime assets, shaders, and dependent DLLs.
- Added or updated icon assets for sketch and mesh tools.

## Documentation And Release Materials

- Added `docs/AxiSim_Software_Manual_v1.04-alpha.tex`.
- Added `docs/AxiSim_Software_Manual_v1.04-alpha.pdf`.
- Added screenshots for sketching, structured meshing, unstructured meshing, residuals, and results.
- Updated README content, feature descriptions, requirements, build instructions, screenshots, license notes, and acknowledgments.
- Added an MIT `LICENSE` file for the original AxiSim source.
- Added explicit documentation that binaries distributed with Gmsh are subject to Gmsh's GPL v2-or-later license terms.

## Fixes And Cleanup

- Cleaned up inspector and shared surface-viewer code.
- Reorganized base surface viewer behavior.
- Fixed bugs in geometry-tab workflows.
- Fixed concentration-solver behavior after the initial concentration implementation.
- Fixed disabled inhibition so it no longer changes the solution.
- Updated `.gitignore` and cleaned build/dependency assumptions around removed libraries.
