# v1.03-alpha Update Log

Baseline used for this summary: `ab51d3b` from 2026-05-21, the commit labeled `v1.02-alpha release`.

Endpoint used: `fc18e8a` from 2026-06-14, the commit where the README identifies the project as `v1.03-alpha`.

Note: there is no `v1.03-alpha` git tag in this repository, so this file treats the README version update as the v1.03-alpha endpoint.

## Highlights

- Reworked the solver from a staggered-grid approach to a collocated finite-volume grid as preparation for future unstructured mesh support.
- Added mesh boundary selection, grouping, saving, loading, and boundary-condition assignment workflows.
- Added support for Dirichlet, Neumann, and fully developed boundary conditions.
- Added an optional energy equation solver and temperature residual plotting.
- Overhauled the mesh GUI with variable grid spacing, easier geometry controls, and boundary-focused tools.
- Improved residual plotting with docking, multiple residual panels, toolbars, and better display options.
- Added console command history and extra console commands, including colormap get/copy actions.
- Began moving the app toward whole-project state management through a `Project` class.

## Solver And Numerics

- Replaced the staggered-grid solver path with a working collocated-grid version.
- Updated SIMPLE algorithm internals for the collocated grid.
- Fixed a SIMPLE divergence issue that affected solver stability.
- Added support for variable grid spacing through biased `dr` and `dz` distributions.
- Added energy equation solving.
- Added support for fully developed inlet conditions.
- Added Dirichlet, Neumann, and fully developed boundary-condition options.
- Added boundary-condition data structures that separate mesh geometry from solver boundary-condition values.
- Added or restored several solver-side features from the earlier staggered-grid solver.
- Added early multigrid-related source/header files.

## Mesh And Boundary Workflow

- Overhauled the mesh GUI to make geometry and grid setup easier to edit.
- Added mesh boundary selection.
- Added selectable boundary edges in the mesh inspector.
- Added boundary edge grouping so selected edges can be organized into named groups.
- Added save/load support for boundary edges.
- Added a prototype boundary-condition maker in the mesh tab.
- Added support for showing created boundary edges in the solver tab.
- Added a function to remove obstacles from the mesh.
- Added `BoundarySegmentGroup` metadata such as orientation and total length.
- Added quality-of-life improvements for selecting mesh boundaries.

## Results, Residuals, And Visualization

- Added residual plot docking support through a new `DockingSpace` helper.
- Added support for displaying multiple residual plots in the solver UI.
- Added residual plot toolbars.
- Added temperature/energy residual display support.
- Added options for flat vs. interpolated result shading.
- Added mesh and inspector updates needed for the new boundary and collocated-grid workflows.
- Added new v1.03-alpha preview screenshots.

## Console And Commands

- Added console command history.
- Added new console commands.
- Added commands for getting and copying colormap values.

## Project, Files, And Structure

- Added a `Project` class to construct and coordinate the mesh, solver, and results components.
- Began moving away from separate run/project pieces toward a unified project workflow.
- Removed the run command as project state became the main workflow.
- Reworked save/load behavior for mesh and solver state.
- Renamed and reorganized file-management code.
- Moved shared data structures into the `structs` folder.
- Reorganized graphics and solver structs into shared headers.

## UI And Cleanup

- Added UI tests and interaction changes.
- Changed the font type.
- Removed the camera dependency from the display constructor.
- Added `BaseSurfaceViewer`, `MeshInspector`, and graphics scene-view organization used by later inspector work.
- Cleaned up GUI/resource manager organization.
- Updated README content and screenshots for the v1.03-alpha preview.

## Known Work-In-Progress Notes From This Range

- Some commits explicitly describe the collocated-grid work as a stepping stone toward unstructured meshes.
- One intermediate commit notes that an outer-wall Neumann boundary-condition attempt was not successful yet.
- The README at the endpoint still points to the older v1.01-alpha manual while showing v1.03-alpha screenshots.
